# NFT MARKETPLACE

To set up the repository and run the marketplace locally, run the below
```bash
git clone https://github.com/DolapoTheCoder/RTW3-Week7-NFT-Marketplace.git
cd RTW3-Week7-NFT-Marketplace
npm install
npm start
```
